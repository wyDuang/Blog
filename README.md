## 技术栈
- 后端：.Net Core 3.0 + EFCore + Dapper + MySql + WebApi + Swagger
- 前端：jQuery + Ajax + Semantic UI
- 部署：CentOS 7.5 + Docker + docker-compose + Azure Pipelines + Nginx
